## Mise en route

### 1-  Se placer dans le bon répertoire
Ouvrez un terminal (CMD, Git Bash ou VS Code) et placez-vous à la racine du projet :
```bash
cd chemin/vers/le/projet
```

### 2-  Vérification de l'environnement
Exécutez le fichier `test.py` afin de vérifier que votre machine dispose de toutes les dépendances nécessaires et que l'environnement fonctionne correctement :
```bash
python test.py
```

### 3- Création d'une branche personnelle
Chaque contributeur doit travailler sur sa branche personnelle.
```bash
git checkout -b votre_nom
git pull
git status
```

**Important :**
- Assurez-vous d'être sur votre branche avant de commencer le développement
- Toutes les modifications doivent être effectuées sur cette branche
